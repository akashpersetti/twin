## Project: Wingman (Self-Evaluating Agentic Co-Worker)
Wingman is a self-evaluating agentic co-worker built with LangGraph, FastAPI, Lambda, and DynamoDB, live at wingman.akashpersetti.com. Akash designed a worker-evaluator state machine in LangGraph where a structured-output evaluator checks worker responses against user-defined criteria and reinjects feedback for up to 5 retry turns, raising answer-validity rates before surfacing a result. He made the agent serverless-safe by reconstructing LangGraph state from DynamoDB on every request instead of an in-memory checkpointer. He integrated 5 tool domains through LangChain ToolNode and bind tools, shipping the full serverless stack via an OIDC pipeline.

## Project: Twin (Streaming AI Digital Twin)
Twin is a streaming AI digital twin built with FastAPI, Next.js, AWS Bedrock (Claude Sonnet), Lambda, S3, and CloudFront, live at akashpersetti.com. Akash built a streaming personal-website agent grounded in a persona assembled at Lambda cold-start from a LinkedIn PDF, career summary, and style guide, and injected into a Bedrock Claude Sonnet system prompt. He shipped a public evals dashboard scoring retrieval quality on labeled queries and cut full-stack deployment to a single command across dev, test, and prod through GitHub Actions and AWS OIDC.

## Project: mcp-second-opinion (Open-Source MCP Server)
mcp-second-opinion is an open-source MCP server published to PyPI at pypi.org/project/mcp-second-opinion. Akash built and published this MIT-licensed MCP server that lets any MCP-aware agent consult rival LLMs mid-conversation, exposing two tools: one to query a single model and one to fan out to all enabled providers in parallel and compare answers. He unified four provider APIs behind a single LiteLLM interface, returning per-call latency, token counts, and cost.
